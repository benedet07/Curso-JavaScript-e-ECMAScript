-Letra, $ ou _

-Não podem conter espaços
-Não podem ser palavras reservadas

5 == 5    True
5 == '5'  True
5 === '5' False     // Verifica se são IDÊNTICOS, mesmo valor e mesmo tipo
5 === 5   True
5 !== '5' True      // Desigual restrito, são tipos diferentes porém mesmo valor interno
